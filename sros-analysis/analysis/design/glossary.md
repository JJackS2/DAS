# Glossary — 핵심 용어 (1줄 정의)

**목적:** 설계·예시·보고서에서 반복 사용되는 용어를 사전 지식 없는 독자가 참조할 수 있도록 1줄로 정의한다.

---

## 분석 절차

| 용어 | 정의 |
|------|------|
| **Phase lock (R0)** | 해당 단계의 필수 산출물이 완료되기 전에는 다음 단계로 넘어가지 않는 규칙. |
| **Evidence rule (R1)** | 데이터에 대한 주장(건수, 분포, 효과 등)은 반드시 스키마·SQL·집계 등 증거를 인용하고, 없으면 UNKNOWN으로 표기하는 규칙. |
| **Time origin** | 분석에서 “시간 0”으로 삼는 사건(예: 첫 구매일, 계정 생성일). |
| **관측 창 (observation window)** | 분석에 포함하는 기간(시작일~종료일). |

---

## 결과·모델

| 용어 | 정의 |
|------|------|
| **Estimand** | 추정하려는 대상(예: “확장 여부에 대한 지역 효과”). 모델은 이 estimand에 맞게 선택한다. |
| **결과 변수 (outcome)** | 설명하려는 변수(예: 확장 여부, 보유 대수). 이항/연속/횟수/시간-이벤트 등 유형에 따라 모델이 달라진다. |
| **공변량 (covariate)** | 결과 변수에 연관될 수 있다고 가정하고 모델에 넣는 변수(예: 지역, 채널, 가입기간). |

---

## 추정·해석

| 용어 | 정의 |
|------|------|
| **오즈비 (OR, odds ratio)** | 로지스틱 회귀에서, 한 공변량이 한 단위 증가할 때 “성공 오즈”가 몇 배가 되는지. 1보다 크면 성공 확률이 증가하는 방향 연관. |
| **Rate ratio** | Poisson/NB 등 횟수 모델에서, 한 공변량이 한 단위 증가할 때 “사건 발생률”이 몇 배가 되는지. |
| **95% CI (신뢰구간)** | 추정치의 불확실성 범위. 반복 표본을 뽑았을 때 100번 중 약 95번 이 구간이 참값을 포함한다고 기대하는 구간. |
| **과분산 (overdispersion)** | 횟수 데이터에서 관측 분산이 모델이 가정하는 분산(Poisson이면 평균=분산)보다 큰 경우. 이때 Negative Binomial 등으로 보정할 수 있다. |

---

## 모델 비교·진단

| 용어 | 정의 |
|------|------|
| **AIC / BIC** | 모델 적합도와 복잡도를 함께 반영하는 지표. 값이 작을수록 선호(동일 데이터 내 비교). |
| **ΔAIC** | 두 모델의 AIC 차이. 차이가 크면 더 작은 AIC를 가진 모델을 선호하는 근거가 된다. |
| **LRT (Likelihood Ratio Test)** | 중첩 모델(한쪽이 다른 쪽의 제약 버전) 간 비교. 검정 통계량과 p-value로 “복잡한 모델이 유의하게 나은지” 판단. |
| **VIF (Variance Inflation Factor)** | 공변량 간 다중공선성 정도. 너무 크면(예: >5~10) 계수 추정이 불안정해질 수 있다. |
| **로버스트니스 (robustness)** | 모델 사양을 조금 바꿨을 때(예: 변수 변환, 구간 변경) 추정이 크게 바뀌지 않는지 확인하는 절차. |

---

## 보고·재현

| 용어 | 정의 |
|------|------|
| **연관 (association)** | 변수 간 통계적 관계. 인과(causation)와는 다르다. 인과는 실험·IV 등 별도 설계가 필요. |
| **재현성 (reproducibility)** | 동일한 코드·데이터·환경으로 실행하면 동일한 결과가 나오도록 문서화(함수·파라미터·시드·패키지 버전)하는 것. |
